== Description ==

A lightweight theme. 

== Installation ==

1. In your site's admin panel, go to Appearance > Themes and click `Add New`.
2. Type "Quattuor" in the search field.
3. Click `Install` and then `Activate` to start using the theme.
4. Navigate to Appearance > Install Plugins and install required plugins.
5. Navigate to Appearance > Demo Import select template to install.
6. Enjoy :)
